Put your wallpapers in this folder (`~/.config/wallpapers` after install).

Supported formats:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

Recommended usage:
- `SUPER+W` for wallpaper menu
- `SUPER+SHIFT+W` for random wallpaper

`~/.config/wallpapers/current` is managed automatically by `wallpaper-apply.sh`.
